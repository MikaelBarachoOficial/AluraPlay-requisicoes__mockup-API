## Tecnologias utilizadas durante o curso
* Javascript
* NodeJS
* Json-server
* HTML
* CSS

## Como mockar a API
Com o Node.js instalado, abra o terminal na pasta do projeto e digite o seguinte:

> npx json-server --watch db.json

db.js é o arquivo que iremos mockar uma API.
